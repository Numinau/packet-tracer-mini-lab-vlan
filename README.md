# packet-tracer-mini-lab-vlan
MiniLab Packet Tracer – VLAN, Trunk, DHCP, Router-on-a-Stick
Mettre en place un mini réseau d’entreprise fonctionnel avec :

✅ Segmentation logique en VLAN
✅ Routage inter-VLAN (Router-on-a-Stick)
✅ Service DHCP centralisé sur le routeur
✅ Tests de connectivité complets
